Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get Info for a Specific Group by id
      morpheus.core.group_info:
        id: 5

    - name: Get Groups Matching Regex Pattern
      morpheus.core.group_info:
        name: ^linux.*$
        regex_name: true

    - name: Get Full Info for all Groups
      morpheus.core.group_info:
        detail: full
